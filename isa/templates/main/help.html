{{ _('About ISA Tool') }}

ISA

{% trans link_attrs='href=https://commons.wikimedia.org/wiki/Commons:Structured_data target=_blank' %}ISA is a multilingual, mobile-friendly tool that simplifies adding structured data to images on Wikimedia Commons, particularly those from Wiki Loves competitions.{% endtrans%}

{{ _('Mobile-First Design') }}

{{ _('Optimized for contributions on smartphones, tablets, and desktop computers') }}

{{ _('Multilingual Support') }}

{{ _('Global participation with support for multiple languages') }}

{{ _('Campaign-Based') }}

{{ _('Organized tagging activities with focused campaigns') }}

{{ _('Point System') }}

{{ _('Track and reward contributions with gamification') }}

{{ _('Wiki Loves Integration') }}

{% trans link_attrs='href=https://commons.wikimedia.org/wiki/Wiki_Loves_Africa target=_blank' %}Originally developed for Wiki Loves Africa, ISA now supports all Wiki Loves competitions and is expanding to serve all media files on Wikimedia Commons.{% endtrans %}
